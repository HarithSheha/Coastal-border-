Database Seed & Login Setup Guide

Document Ref CBS-DEV-001
System Laravel 13 + MySQL
Frontend React + Vite (localhost:5173)
Backend localhost:8000
⚠ Complete all steps in order. MySQL must be running before executing any artisan commands, otherwise the commands will fail with a connection error.

1 Start MySQL (XAMPP)

Open the XAMPP Control Panel

Navigate to your XAMPP installation and run the control panel, or use the start script directly:

C:\xampp> mysql_start.bat

Verify MySQL is running

Open a new terminal and confirm port 3306 is listening:

PS> netstat -ano | findstr ":3306"
Expected output
TCP    0.0.0.0:3306           0.0.0.0:0               LISTENING

2 Open the Backend Directory

Navigate to the Laravel project folder

Open PowerShell or Command Prompt and change into the backend directory:

PS> cd "C:\Users\dulas\Desktop\SEPTEMBER KISW MONTHLY TEST\A2 HISTRY PASTPAPER\Documents\Dev\costal-back\Coastal-Border-backend"

3 Run the Database Migration

Apply all pending migrations

This creates the users, zones, sensors, live_reports, and all other required tables:

PS> php artisan migrate
Expected output
INFO  Running migrations.
 2026_07_06_000001_add_auth_fields_to_users_table .. DONE

If you see "Nothing to migrate", the migrations already ran — that is fine, proceed to Step 4.

4 Seed the Database

Seed users only (recommended first run)

Creates the four login accounts without overwriting existing zone/sensor/report data:

PS> php artisan db:seed --class=UserSeeder
Expected output
INFO  Seeding database.
 Users seeded:
  admin   / admin123   (Administrator)
  mwangi  / officer123 (Officer)
  achieng / officer123 (Officer)
  otieno  / officer123 (Officer)

Seed all data (fresh install only)

Seeds users, zones, sensors, live reports, and sensor readings. Use this only on a fresh database — it will duplicate records if run again:

PS> php artisan db:seed

5 Login Credentials

Use any of the accounts below on the login screen at http://localhost:5173.

Username Password Role Full Name
admin admin123 Administrator System Administrator
mwangi officer123 Officer Officer Mwangi
achieng officer123 Officer Sgt. Achieng
otieno officer123 Officer Cpl. Otieno

6 Start Both Servers

Terminal 1 — Laravel backend

PS> php artisan serve

Terminal 2 — React frontend

PS> npm run dev

Open the dashboard in your browser

🌐
http://localhost:5173
The login screen will appear. Enter username and password from the table above.
